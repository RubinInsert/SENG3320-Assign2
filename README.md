# SENG3320 Assignment 2

## Compiling and Executing the Program
The source code for this project can be found in `/src/main/java/com/rubinin/seng3320assign2`.
The main file is called `PacketFuzzer.java`.
After installing a working version of Java, the project can be run with the command:
```
java src\main\java\com\rubinin\seng3320assign2\PacketFuzzer.java
```

## Crash Corpus
The crash corpus can be found under `/crashes`

## Java Version Used
This project was tested on the 2025 LTS version of java (Java 25).
